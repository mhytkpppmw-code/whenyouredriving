# When You're Driving

Fill in the blanks and submit your variation of the song:

**When you're driving in your _____ and you _____, diarrhea, 💨💨, diarrhea.**

## Local development

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy

Connect this repo to [Vercel](https://vercel.com). Submissions are stored in `data/submissions.json` (works on Vercel for reads; new submissions persist until the next redeploy unless you add a database later).

## Stack

- Next.js 15
- React 19
- Tailwind CSS
